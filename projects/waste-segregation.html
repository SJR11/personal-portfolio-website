Deep Learning Based Real Time Waste Segrgation Using Robotic Arm

This project automates the waste segregation process using deep learning and robotics. Waste materials are placed on a conveyor belt, identified in real-time using a YOLO-based object detection model, and sorted by a robotic arm controlled with Arduino and Raspberry Pi.
