Robo Race

This project showcases a high-speed line-following robot built for racing competitions. The robot is engineered with PID-controlled precision and custom wheel alignment to handle sharp turns and high-speed straight runs. It features a lightweight frame and powerful motors, optimized for balance between speed and control.
