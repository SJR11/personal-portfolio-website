Surveillance Rover

This project features a smart surveillance rover designed for remote monitoring in inaccessible or dangerous environments. Equipped with live video streaming, DHT11 sensor, and wireless control, the rover can be navigated remotely to gather real-time data and ensure safety in critical operations.

Technologies Used

  • 📷 Camera: Pi Camera for live video
  • 📡 Control: RF Module
  • 🛠️ MCU:Electronic Speed Controller
  • 🪙 Power: LiPo Battery 12V
  • 🧠 Features: Environmental monitoring, real-time streaming, remote operation
